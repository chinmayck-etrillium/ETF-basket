# ETF Portfolio Analytics Dashboard

A professional ETF Portfolio Analytics Dashboard built in Excel for evaluating diversified ETF portfolios using Modern Portfolio Theory (MPT), risk-adjusted return metrics, and portfolio analytics.

This project analyzes an ETF basket using historical daily return data and calculates key portfolio management metrics such as expected return, volatility, Sharpe ratio, beta, covariance matrix, and diversification statistics.

---

# Features

## Portfolio Analytics

* Expected Portfolio Return
* Portfolio Volatility (Risk)

* Beta Calculation
* Correlation Matrix

# Key Learnings From This Project

* Portfolio risk is not a weighted average of individual risks.
* Diversification significantly reduces portfolio volatility.
* Correlation plays a major role in portfolio construction.
* Risk-adjusted returns are more important than standalone returns.
* Portfolio management is primarily about risk management.
